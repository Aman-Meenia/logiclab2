# Count Number Of Subarray With Sum Less than Or Equal To Zero

&nbsp;

#### Given an array arr[] of size n containing 0 and 1 only. The problem is to count the subarrays with sum less than 0.

&nbsp;

**Example 1:**

&nbsp;

**Input**

```
arr = [-1, -2, 1, 2 ,-3]
```

&nbsp;

**Output**

```
11
```

&nbsp;

**Example 2:**

&nbsp;

**Input**

```
arr = [3,-1,3,9,-8,1,-12,1,-3,4,-1]
```

&nbsp;

**Output**

```
42
```

&nbsp;
**Example 3:**

&nbsp;

**Input**

```
arr = [1,1,1,-1,1,1]
```

&nbsp;

**Output**

```
3
```

&nbsp;

### Constraints

&nbsp;

- 1 <= arr.length <= 10^5

  &nbsp;

- -10^5 <= arr[i] <= 10^5
