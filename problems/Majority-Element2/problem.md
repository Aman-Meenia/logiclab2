# Majority Element 2

&nbsp;

#### Given an integer array of size n, find all elements that appear more than ⌊ n/3 ⌋ times.

#### You may assume that the array is non-empty and the majority element always exist in the array.

&nbsp;

**Example 1:**

&nbsp;

**Input**

```
arr = [3, 2, 3]
```

&nbsp;

**Output**

```
3
```

&nbsp;

**Example 2:**

&nbsp;

**Input**

```
arr = [1]
```

&nbsp;

**Output**

```
1
```

&nbsp;
**Example 3:**

&nbsp;

**Input**

```
arr = [3, 3]
```

&nbsp;

**Output**

```
3
```

&nbsp;

### Constraints

&nbsp;

- 1 <= arr.length <= 10^5

  &nbsp;

- -10^9 <= arr[i]= 10^9

  &nbsp;
