# Majority Element

&nbsp;

#### Given an array arr. Find the majority element in the array. If no majority exists, return -1. A majority element in an array is an element that appears strictly more than arr.size() / 2 times in the array.

&nbsp;

**Example 1:**

&nbsp;

**Input**

```
arr = [1, 1, 2, 1, 3, 5, 1]
```

&nbsp;

**Output**

```
1
```

&nbsp;

**Example 2:**

&nbsp;

**Input**

```
arr = [3, 3, 4, 2, 4, 4, 2, 4]
```

&nbsp;

**Output**

```
-1
```

&nbsp;
**Example 3:**

&nbsp;

**Input**

```
arr = [3]
```

&nbsp;

**Output**

```
3
```

&nbsp;

### Constraints

&nbsp;

- 1 <= arr.length <= 10^5

  &nbsp;
